# Tienda Digital Scrum

Aplicación académica de tienda digital para una empresa única. El proyecto se construye incrementalmente con commits por requisito, usando React, FastAPI y MySQL.

## Stack

- Frontend: React, Vite, React Router, Axios y CSS.
- Backend: Python, FastAPI, SQLAlchemy, Pydantic, JWT y Passlib/bcrypt.
- Base de datos: MySQL con scripts `schema.sql` y `seed.sql`.
- Ejecución local: Docker Compose para MySQL, Uvicorn para backend y Vite para frontend.
